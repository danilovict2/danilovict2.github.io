Online Chess Platform

This project is a comprehensive clone of the chess.com platform, developed as a school project. The backend is powered by Symfony, while the frontend utilizes Vue.js and Bootstrap for a seamless user experience. Real-time chess games against other online players are enabled using Mercure. Additionally, the project features gameplay against the Stockfish engine, with adjustable Elo settings for varying difficulty levels.

Pinpoint

This project replicates the Geoguessr game and was developed for a Hackathon, where it placed 7th out of 25+ entries. Built using Vue.js, it features three engaging game modes:
1. Standard Pinpointing - Guess the location for maximum points.
2. Country Guessing - Identify the country for high scores.
3. Flag Guessing - Match flags to countries.
Note: The project is in Serbian as it was made for a Serbian Hackathon.

E-commerce Website

This project utilizes Symfony for the backend and Vue.js with TailwindCSS for the frontend. It features an admin panel built with EasyAdmin, offering various statistics and options to add, update, or remove products. The checkout system is integrated with Stripe for secure and efficient transactions.

Realtor Clone

This project utilizes Symfony 6 for the backend and Vue.js with TailwindCSS for the frontend. It features a real estate listing platform with core functionalities for browsing property listings. Real-Time Chat

This project leverages Symfony 6 for the backend and Vue.js for the frontend to build a real-time chat application. The app features instant messaging capabilities with WebSocket integration for real-time communication. Users can send and receive messages in real-time. The frontend is styled with Bootstrap. The application supports user authentication and provides a secure environment for messaging and user interactions.
